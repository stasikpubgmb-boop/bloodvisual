# BloodVisual

A Minecraft 1.21.4 Fabric mod by Veron

## Description

BloodVisual adds visual blood effects to Minecraft.

## Requirements

- Minecraft 1.21.4
- Fabric Loader 0.16.9+
- Fabric API 0.110.0+
- Java 21+

## Installation

1. Install Fabric Loader
2. Download the mod JAR file
3. Place it in your `.minecraft/mods` folder
4. Launch Minecraft with the Fabric profile

## Building

```bash
./gradlew build
```

The built JAR will be in `build/libs/`
